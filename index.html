🛠️ Setup Guide

1 · Install Ollama
curl -fsSL https://ollama.ai/install.sh | sh
ollama pull llama3
ollama pull deepseek-coder
2 · Install Python packages
pip install fastapi uvicorn ollama
3 · Run the server
python server.py
Then open http://localhost:8000 in Chrome or Edge.
🎤 Voice — No installs needed!
Voice uses the Web Speech API built into Chrome & Edge. No Whisper, no ffmpeg, no microphone libraries.
Just click Start Recording, speak, click Stop, edit if needed, then Submit.
